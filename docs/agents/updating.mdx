## Update an agent

When an update is available, open the server in the control plane and click **Update**. Confirm the target version, then click **Queue upgrade**.

The control plane sends a work item over the authenticated agent channel. The agent:

1. Downloads the target version of the agent binary from GitHub releases.
2. Verifies the SHA256 checksum.
3. Replaces the existing binary.
4. Restarts itself after installation.

No configuration or dependencies are changed during an update.

## Manual update

Download the binary for your architecture:

```bash
# For x86_64
curl -fsSL https://github.com/techulus/cloud/releases/latest/download/agent-linux-amd64 \
  -o /usr/local/bin/techulus-agent
chmod +x /usr/local/bin/techulus-agent

# For arm64
curl -fsSL https://github.com/techulus/cloud/releases/latest/download/agent-linux-arm64 \
  -o /usr/local/bin/techulus-agent
chmod +x /usr/local/bin/techulus-agent
```

Restart the service:

```bash
sudo systemctl restart techulus-agent
```

## Verify

Check that the agent is running after the update:

```bash
sudo systemctl status techulus-agent
sudo journalctl -u techulus-agent -n 20
```
